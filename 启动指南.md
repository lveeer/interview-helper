# 智能面试提升系统 - 启动指南

## 项目简介

基于 AI 的面试辅助平台，帮助求职者进行简历优化、岗位匹配、模拟面试和知识管理。

## 环境要求

- Node.js >= 16.x
- Python >= 3.8
- Docker（用于 PostgreSQL 数据库）

## 快速启动

### 启动顺序

**重要：必须先启动后端，再启动前端！**

```
后端服务 (端口 8000) → 前端服务 (端口 5173) → 浏览器访问
```

---

### 步骤 1：启动后端

打开终端，执行以下命令：

```bash
# 进入后端目录
cd backend

# 创建虚拟环境（首次运行）
python -m venv venv

# 激活虚拟环境
# Linux/Mac:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 安装依赖（首次运行）
pip install -r requirements.txt

# 配置环境变量
cp .env.example .env
# 编辑 .env 文件，填写必要配置

# 启动数据库
docker-compose up -d

# 初始化数据库
alembic upgrade head

# 启动后端服务
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

**启动成功标志：**
- 终端显示 `INFO: Uvicorn running on http://0.0.0.0:8000`
- 访问 `http://localhost:8000/docs` 可查看 API 文档

---

### 步骤 2：启动前端

打开新终端，执行以下命令：

```bash
# 进入前端目录
cd frontend

# 安装依赖（首次运行）
npm install

# 创建环境变量文件
echo "VITE_API_BASE_URL=http://localhost:8000/api" > .env.development

# 启动前端服务
npm run dev
```

**启动成功标志：**
- 终端显示 `Local: http://localhost:5173/`
- 浏览器访问 `http://localhost:5173` 可看到登录页面

---

### 步骤 3：验证服务

1. 访问 `http://localhost:8000/docs` 确认后端 API 正常
2. 访问 `http://localhost:5173` 确认前端页面正常
3. 注册/登录账号，测试基本功能

## 常见问题

### Q: 前端无法连接后端？

1. 检查后端是否启动：访问 `http://localhost:8000/docs`
2. 检查端口是否被占用：`lsof -i :8000`（Linux/Mac）
3. 确认 `.env.development` 文件中 `VITE_API_BASE_URL` 配置正确

### Q: npm install 失败？

```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 端口被占用？

```bash
# Linux/Mac
lsof -ti:8000 | xargs kill -9

# Windows
netstat -ano | findstr :8000
taskkill /PID <进程ID> /F
```

### Q: 虚拟环境激活失败？

- **Linux/Mac:** `source venv/bin/activate`
- **Windows CMD:** `venv\Scripts\activate.bat`
- **Windows PowerShell:** `venv\Scripts\Activate.ps1`
